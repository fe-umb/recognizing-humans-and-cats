A saída (imagens) são sempre a combinação da palavra "out" + número. O número é inserido de acordo com o número da imagem original. Caso deseje adicionar mais imagens, não esqueça de trocar o valor da variável "n_images". Atualmente, o projeto conta com 20 imagens.

Não se esqueça de alterar o diretório no início do projeto.

=============================

The output (images) are always the combination of the word "out" + number. The number is entered according to the original image number. If you want to add more images, don't forget to change the value of the variable "n_images". Currently, the project has 20 images.

Don't forget to change the directory at the beginning of the project.
